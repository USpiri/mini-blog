Angular es un popular framework de desarrollo de aplicaciones web que se utiliza para construir aplicaciones de una sola página (SPA). Para garantizar que nuestro proyecto Angular sea mantenible, escalable y eficiente, es importante seguir buenas prácticas en el desarrollo. A continuación, se presentan algunas pautas y recomendaciones para escribir código limpio y estructurado en Angular.

## Objetivo

Este documento tiene como objetivo proporcionar una guía completa de buenas prácticas en Angular. Abordaremos desde la estructura de proyectos hasta las prácticas recomendadas para la creación de componentes, servicios, enrutamiento, gestión de estado y pruebas.

## Angular CLI

El CLI (Command Line Interface) de [Angular](https://cli.angular.io/) es una herramienta esencial para desarrollar aplicaciones en Angular. Proporciona una interfaz de línea de comandos que facilita la creación, el desarrollo, la compilación y el despliegue de proyectos Angular. Dado que esta es una herramienta oficial de Angular, la primer recomendación es hacer uso de ella. A continuación algúnos comandos útiles:

| Comando                                      | Descripción                               |
| -------------------------------------------- | ----------------------------------------- |
| npm install -g @angular/cli                  | Instalar Angular CLI                      |
| ng new \<app-name\>                          | Crear un nuevo proyecto Angular           |
| ng generate component \<component-name\>     | Generar un nuevo componente               |
| ng serve                                     | Ejecutar la aplicación localmente         |
| ng generate module \<module-name\> --routing | Generar un nuevo modulo con routing       |
| ng add @\<dependency-name\>                  | Agregar biblioteca con Angular Schematics |
| ng update @angular/cli                       | Actualiza el CLI                          |
| ng build --prod                              | Compila el proyecto para producción       |
