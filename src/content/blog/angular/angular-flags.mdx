## Generate

Flags para el comando `generate`. Más en la [documentación de Angular](https://angular.io/cli/generate)

| Flag                | Description                                                           |
| ------------------- | --------------------------------------------------------------------- |
| `--skip-tests`      | No crea el fichero de testing para el componente                      |
| `--inline-style`    | Estilos inline (dentro de component.ts). No crea el archivo de estilo |
| `--inline-template` | Template inline (dentro de component.ts). No crea el archivo .html    |
| `--standalone`      | Genera un componente standalone                                       |

## Serve

Flags para el comando `serve`.

| Flag             | Description                                            |
| ---------------- | ------------------------------------------------------ |
| `--o` / `--open` | Abre el navegador al ejecutar el entorno de desarrollo |
